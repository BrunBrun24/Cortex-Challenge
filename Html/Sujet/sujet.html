Couleur

Quel mot est écrit dans sa bonne couleur ?

Particularités :

  • Les clefs du dictionnaire colors peuvent être écrites en anglais et en français.
  • Les valeurs du dictionnaire colors sont un code couleur tkinter correspondant à la couleur écrite en anglais.
  • Le nombre de couleurs de la carte est variable.
  • Pour gagner il faut renvoyer la clef du dictionnaire colors correspondant à la même couleur que sa clef.
  • 20 couleurs doivent être considérées : argent, beige, blanc, bleu, corail, indigo, jaune, lavande, magenta, marron, mauve, noir, olive, or, orange, orchidée, rose, rouge, saumon et vert.

    • Données (JSON)

    {
    "code" : "CO-001",
    "type" : "couleur",
    "colors" : {
        "black": "red",
        "blue" : "green",
        "mauve": "black",
        "green" : "purple",
        "white" : "blue",
        "orange" : "yellow",
        "pink" : "pink",
        "yellow" : "white",
        "red" : "orange"
    }
}

        Réponse : "vert"
