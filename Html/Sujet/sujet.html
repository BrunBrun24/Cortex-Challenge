Couleur

Quel mot est écrit dans sa bonne couleur ?

Particularités :

  • Les clefs du dictionnaire colors peuvent être écrites en anglais et en français.
  • Les valeurs du dictionnaire colors sont un code couleur tkinter correspondant à la couleur écrite en anglais.
  • Le nombre de couleurs de la carte est variable.
  • Pour gagner il faut renvoyer la clef du dictionnaire colors correspondant à la même couleur que sa clef.
  • 20 couleurs doivent être considérées : argent, beige, blanc, bleu, corail, indigo, jaune, lavande, magenta, marron, mauve, noir, olive, or, orange, orchidée, rose, rouge, saumon et vert.

Données (JSON)

{
    "code" : "CO-001",
    "type" : "couleur",
    "colors" : {
        "black": "red",
        "blue" : "green",
        "mauve": "black",
        "green" : "purple",
        "white" : "blue",
        "orange" : "yellow",
        "pink" : "pink",
        "yellow" : "white",
    }
}
        Réponse : "vert"

Réflexion

En partant de la lampe torche, quel est le numéro que le faisceau lumineux va atteindre en se réfléchissant sur les miroirs blancs ?

Particularités :

  • La taille de la map, les numéros et l'emplacement de la lampe sont fixes.
  • Le nombre de miroir est variable.
  • Les réflexions possibles sont données ici.

Données (JSON)

{
    "code" : "RE-001",
    "type" : "réflexion",
    "map" : [
        [" ",  1, 2, 3, 4, ""],
        [15, "", "", "", "", 5],
        [14, "", "/", "", "/", 6],
        [13, "", "", "", "", 7],
        [12, "", "", "", "", 8],
        ["", 11, "L", 10, 9, ""]
    ]
}

        Réponse : 15

Calcul

Quels nombres sont à additionner afin d’arriver au résultat central ?

Particularités :

  • Le nombre de nombre disponible pour le calcul est fixe (6).
  • La valeur des nombres à additionner est un nombre entier positif qui peut être supérieure au résultat central.
  • Les 6 nombres disponibles ont des valeurs différentes.
  • Le résultat doit être donné sous forme de chaîne de caractères où les nombres apparaissent dans l'ordre croissant et séparés par le signe +.

Données (JSON)

{
    "code" : "CA-001",
    "type" : "calcul",
    "result" : 14,
    "numbers" : [2, 5, 11, 7, 1, 4]
}                      

Réponse : "1+2+11" OU "2+5+7" OU "1+2+4+7"

Fréquence

Quel est le mot le moins représenté ?

Particularités :

  • La taille de words est totalement variable.
  • Un seul mot est le moins représenté.
  • N'importe quel mot doit être considéré. La liste des mots n'est donc pas connue à l'avance.

Données (JSON)

{
    "code" : "FR-001",
    "type" : "fréquence",
    "words" : [
        ["cow",  "dog", "dog", "cat", "chicken", "firefox"],
        ["dog", "cow", "cat", "chicken", "cat", "cat"],
        ["chicken", "dog", "cat", "chicken", "firefox", "chicken"],
        ["cat", "cow", "dog", "cat", "dog", "cow"],
        ["firefox", "cat", "dog", "cow", "dog", "chicken"],
        ["cow", "cow", "dog", "chicken", "cat", "dog"]
    ]
}

        Réponse : "firefox"

Manquant

Quel nombre manque-t-il à une des deux suites numériques ?

Particularités :

  • La taille de numbers est variable.
  • 2 couleurs sont utilisées : R pour rouge et B pour bleu.
  • Le nombre d'éléments dans chaque couleur est variable.
  • Il ne peut y avoir qu'un seul manquant dans une seule couleur.
  • Le résultat doit être un nombre correspondant à la valeur manquante dans une des deux suites.

Données (JSON)

{
    "code": "MA-001",
    "type": "manquant",
    "numbers": ["1R", "4R","5R", "4B", "7B", "8B", "2R", "3R", "5B"]
}

        Réponse : 6
